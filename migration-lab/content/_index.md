I help teams ship reliable software faster by designing resilient platforms, automating delivery, and making systems observable.

## How I work

- Diagnose the real constraint (latency, reliability, deploy risk, toil).
- Design small, testable changes (architecture + rollout plan).
- Automate and document the path to production (CI/CD, IaC, runbooks).
- Measure impact (SLOs, error budgets, cost signals) and iterate.

## What I can help with

- Platform hardening: incident readiness, backups, security baselines.
- CI/CD: safer releases, faster pipelines, reproducible builds.
- Observability: logs/metrics/traces that actually answer questions.
- Cloud infrastructure: Terraform, containerization, networking, IAM.
- Python services: pragmatic microservices, async workloads, API reliability.
